# CPP exercises

Simple exercises for learning [C++ 23](https://isocpp.org/).


## Requirements

 - [GCC](https://gcc.gnu.org/) **13.3.x** and above
 - [vcpkg](https://vcpkg.io/en/) **2026-05-27** and above
 - [Cmake](https://cmake.org/download/) **3.28.x** and above

## Recommended tools
 - [just](https://github.com/casey/just) **1.47.x** and above


## Getting Started

### Clone the repo and make it yours:

```bash
$ git clone git@gitlab.com:diniremix/cpp-examples.git
```

### Build

```bash
cd cpp-examples
```

```bash
just build
```
